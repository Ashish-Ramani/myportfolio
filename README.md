# Ashish Ramani - 3D Portfolio

A modern, interactive 3D portfolio website showcasing my work as a Data Engineer. Built with React, TypeScript, Vite, and GSAP animations.

## 🚀 Features

- **Modern 3D Animations**: Smooth scroll-triggered animations using GSAP
- **Responsive Design**: Fully responsive across all devices
- **Interactive Sections**: Hero, Projects, Skills, Experience, About, Process, Contact, and Footer
- **Glass Morphism UI**: Beautiful glassmorphic design elements
- **Smooth Scrolling**: Custom snap scrolling with pinned sections
- **TypeScript**: Fully typed for better development experience
- **Tailwind CSS**: Utility-first CSS framework with custom theme

## 🛠️ Tech Stack

- **React 19** - UI library
- **TypeScript** - Type safety
- **Vite** - Build tool and dev server
- **GSAP** - Animation library
- **Tailwind CSS** - Styling
- **Radix UI** - Accessible component primitives
- **Lucide React** - Icon library

## 📦 Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 🌐 Deployment

This project is configured for GitHub Pages deployment. The build output is in the `dist` folder.

### GitHub Pages Setup

1. Push your code to GitHub
2. Go to repository Settings → Pages
3. Select source: `gh-pages` branch or `main` branch `/dist` folder
4. Your site will be live at `https://[username].github.io/[repository-name]`
